email-card-sender 💌

Send tidy HTML “Diego card” emails via gog, with CC rules and Outlook‑friendly layouts.

This skill owns the actual sending of emails on Drew’s behalf when we want the Diego card look – rounded box, header, well‑spaced paragraphs, and a friendly sign‑off. It standardises how we call gog gmail send, keeps the HTML compact, and always CCs Drew for external emails.

Show more about email-card-sender

Use this skill whenever an email should use the full Diego HTML card style or when we’re sending short, well‑structured HTML communications where readability and vibe matter. It handles quoting safely, uses proven card templates, and ensures every external email on Drew’s behalf copies him in.

diego-site-updater 🌐

Safely edits and deploys the Diego site + mini‑sites.

Handles HTML/CSS edits, new sections, nav links, and git flows for diego-site and related repos. It’s the main muscle behind the Webmaster builder agent – keeping layout consistent, wiring new pages, and committing changes cleanly.

Show more about diego-site-updater

This skill is the preferred way to touch the Diego site: it reads existing HTML/CSS, applies small, targeted changes, and then runs the git add/commit/push flow inside the correct repo. It’s how we safely evolve hero sections, nav bars, NRL pages, and mini‑sites without wrecking layout.

gog 🎮

Google Workspace CLI: Gmail, Calendar, Drive, Docs, Sheets, Contacts.

Powers email and calendar work: searching inbox, sending HTML emails, managing events, and interacting with Drive/Docs/Sheets when needed. Emailer leans heavily on this for comms and scheduling.

Show more about gog

We use gog to do nearly everything with Google Workspace from the command line: gog gmail for inbox triage and sends, gog calendar for events, and gog docs/sheets/drive when documents or spreadsheets need to be created or updated programmatically.

social-maestro 🎭

Keeps Drew’s social life humming with witty, warm messages and nudges.

Helps draft and coordinate catch‑ups, check‑ins, and little social nudges. Focused on tone, timing, and making sure Drew doesn’t lose track of people.

Show more about social-maestro

Social maestro is the little conductor for Drew’s social world – it remembers context about friends, prefers warm, slightly playful language, and can turn vague “we should catch up” intentions into concrete, nicely worded messages with options.

nrl-tipping 🏉

Generates round tips and writes them into Diego’s NRL tipping pages.

Uses fixtures, form and injuries to produce weekly NRL tips, confidence levels, and short explanations, then updates the NRL tipping section on this site with winners and commentary.

Show more about nrl-tipping

This skill handles the full NRL tipping flow: reading current round fixtures, making picks, assigning confidence levels, and writing the output into the static tipping pages with ✅/❌ markers once games are complete.

telegram-media-saver 📁

Copies inbound Telegram media into repo media/ folders.

Moves photos and files from OpenClaw’s shared Telegram inbound directory into the right media/ folders (e.g. diego-site/media) so they can be used on sites without manual file shuffling.

Show more about telegram-media-saver

When Drew sends pictures or files into Telegram, this skill is responsible for relocating them into project repos with sensible filenames and paths, so pages can reference them directly without digging through the raw inbound folder.

summarize 🧾

Turns long text, URLs or files into digestible summaries.

Great for news articles, transcripts, and longer docs. Produces structured summaries or bullet points that Drew can use for decision‑making, emails, or quick briefings.

Show more about summarize

Summarize is the generic "shrink this wall of text" tool – ideal for meeting notes, policy documents, or multi‑article news briefings. It can output bullet lists, short narratives, or structured sections depending on what Drew needs.

github-upload 🚀

Takes a local directory and turns it into a new GitHub repo.

Handles the initial git init, remote creation, and first push so small projects can be published quickly without hand‑rolling git commands every time.

Show more about github-upload

This skill wraps the boilerplate needed to publish a folder: initialize git, create a GitHub repo with the desired visibility, set the remote, and push. Great for turning local experiments into shareable repos with minimal ceremony.

weather 🌦️

Quick current conditions and forecasts via wttr.in or Open‑Meteo.

Answers "what’s the weather like?" and short‑range forecast questions without extra setup – useful when we’re planning outings or deciding whether to reshuffle events.

Show more about weather

Weather uses lightweight HTTP APIs (like wttr.in or Open‑Meteo) to fetch current conditions and simple forecasts, so we can quickly decide whether to move dinners inside, reschedule walks, or bring umbrellas without opening a weather app.

xurl 🐦

Talks to the X / Twitter API for posts, reads, DMs, and more.

Used when we want to post, reply, search, or manage things on X (Twitter) via the authenticated API instead of the web UI.

Show more about xurl

xurl wraps authenticated calls to the X (Twitter) API so we can post tweets, reply, quote‑tweet, search timelines, send DMs, and manage followers without touching the browser. It’s the bridge between Diego and Drew’s X presence.

openai-whisper 🎙️

Local speech‑to‑text for audio files (no external API key).

Runs Whisper locally to transcribe recordings into text, which can then be summarised, turned into emails, or added to logs.

Show more about openai-whisper

This skill relies on the Whisper CLI installed on the host machine to convert audio (voice notes, meeting recordings, quick memos) into text. Once transcribed, other skills like summarize or email-card-sender can take over.

video-frames 🎞️

Extracts frames or short clips from videos using ffmpeg.

Useful for pulling stills, quick previews, or tiny clips from longer videos without opening a full editor.

Show more about video-frames

video-frames wraps ffmpeg invocations to grab key frames or short segments from larger video files. Handy for creating thumbnails, reference stills, or short clips to share.

blogwatcher 📡

Monitors blogs and RSS/Atom feeds for updates.

Keeps an eye on selected feeds and surfaces new posts so Drew doesn’t have to manually check a dozen sites for fresh content.

Show more about blogwatcher

blogwatcher uses a small CLI to poll RSS/Atom feeds and record changes. When something new appears, it can surface the update in chat or logs, so Drew sees important posts without manual checking.

github / gh-issues 🧑‍💻

GitHub operations via the gh CLI: issues, PRs, CI, and reviews.

Used for checking PR status, CI runs, creating/commenting on issues, and doing lightweight code review or log inspection directly from the command line.

Show more about github / gh-issues

The GitHub skills sit on top of the gh CLI, handling things like pulling issue lists, watching CI, inspecting PRs, and replying to review comments. They’re the bridge between local code changes and GitHub’s workflow.

self-improvement 🧠

Logs mistakes, corrections and better patterns so future‑Diego is less dumb.

Captures learnings when commands fail, tools misbehave, or Drew corrects my output, and updates internal notes so we don’t repeat the same errors forever.

Show more about self-improvement

self-improvement is the meta skill: it records what went wrong, what we learned, and better ways to do recurring tasks. Over time, that becomes a playbook that future‑Diego uses to avoid repeating the same mistakes.
